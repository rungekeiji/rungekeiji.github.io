Guide:

  • You can add a task by entering anything on the field and press the button
  • You can add a sub-task by entering anything on the field and press the button on the associated main task
  • A due date can be associated with a main task by pressing on the calender icon
  • The main task can be color coded individually by pressing on the circle to the left of the main task
  • The main task along with any sub-task can be marked as completed and deleted altogether
